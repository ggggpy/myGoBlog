Golang 基础 -- Map(映射)

map是一个无序得key,value类型; map 和 selice 一致, 属于引用类型; 不初始化时 == nil

声明

mapName := map[keyType]valueType     //未初始化
mapName := map[keyType]valueType{    //声明并赋值
    key1 : value1,
    key2 : value2,
}

mapName := make(map[keyType]valueType, [cap])   //申请内存空间
